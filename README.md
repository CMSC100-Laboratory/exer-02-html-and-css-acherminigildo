Steps on how to create a website using GitHub page:

    1.) Create a repository in your local computer. Add the README, HTML, and CSS files into the repository. Push the repository into GitHub. With the current exercise, the link for the GitHub assignment was given.

    2.) Change the GitHub repository as public visibility in the repository's general settings. 

    3.) In GitHub settings pages tab, change Branch from 'none' to 'main'. Wait for the link to be created. 

Congrats! The website has now been created with GitHub page.
